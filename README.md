🚀 BG-Project

A modern full-stack web application built to demonstrate real-world development skills using MERN stack + AI integration concepts.

📌 Overview

BG-Project is designed to showcase scalable backend architecture, clean frontend UI, and practical problem-solving using modern technologies.
It focuses on performance, modular code structure, and real-world usability.

✨ Features
🔐 Authentication & Authorization (JWT based)
📊 Dynamic Dashboard
🌐 RESTful API integration
⚡ Fast and responsive UI
🗂️ Modular backend architecture
📦 Database integration (MongoDB)
🧠 Scalable structure for future AI/ML integration

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/charitra-21/BG-Project.git
cd BG-Project
2️⃣ Install dependencies
For backend
cd server
npm install
For frontend
cd client
npm install
3️⃣ Setup Environment Variables

Create a .env file in the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
4️⃣ Run the project
Start backend
cd server
npm run dev
Start frontend
cd client
npm run dev
🚀 Usage
Open browser: http://localhost:5173
Register/Login
Explore dashboard and features
